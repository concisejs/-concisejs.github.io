安装及使用

环境准备#

node 环境#

ConciseJS 需要较新的 node 环境，请确保 node 版本在 12.0.0 或以上。

PS I:\Code> node -v
v12.18.4

推荐使用 node 版本管理工具 nvm（适用于 Mac/Linux ）或 nvm-windows（适用于 Windows ）来管理 node 版本。

PS I:\Code> nvm list
* 12.18.4 (Currently using 64-bit executable)
8.6.0

使用内网源#

公司内网搭建了私有的 npm 仓库 私有NPM ，使用内网源可以大大提高第三方依赖包的下载速度。

ConciseJS 发布在了内网仓库中，所以在使用前需要先将 npm 官方源切换为公司内网源：

PS I:\Code> npm config set registry http://192.168.0.94:7002

安装#

安装 ConciseJS 项目有两种方式：使用脚手架安装，或者直接使用 npx 安装（推荐）。

使用脚手架安装#

使用脚手架安装时，需要先全局安装脚手架工具：

# 全局安装
PS I:\Code> npm install -g @concisejs/cli
# 查看脚手架使用帮助
PS I:\Code> concise -h
Usage: concise [options] <command>
Options:
-v, --version output the current version
-h, --help display help for command
Commands:
create <project-name> create a new project
Run concise <command> --help for detailed usage of given command.

然后使用脚手架安装 ConciseJS 项目：

PS I:\Code> concise create myAp

使用 npx 安装（推荐）#

npm 从 5.2 版本开始，增加了 npx 命令。

推荐使用 npx 安装 ConciseJS 项目：

PS I:\Code> npx @concisejs/cli create myApp

使用#

在创建 ConciseJS 项目时，会默认执行 Git 初始化和安装依赖的操作，所以在项目创建完成后，可以直接进入开发和代码提交。

npm start 或 yarn start#

切换工作目录，直接运行 npm start 或 yarn start 命令启动开发模式下的应用程序：

PS I:\Code> cd myApp
PS I:\Code\myApp> yarn start

在开发模式下，编辑代码，页面将重新加载。

npm run build 或 yarn build#

打包生成部署代码，会按照公司前后端约定的路径进行打包，打包到 dist/Web/项目名/ 下（如人脸 Face 项目，路径为： dist/Web/Face/），同时将 release 文件写入 dist/Web/项目名/static/ 目录下：

PS I:\Code\myApp> yarn build

首次提交代码到远程仓库#

$ git add .
$ git commit -m 'Initial commit'
$ git remote add origin <git-url>
$ # For example:
$ # git remote add origin http://192.168.1.36:3000/huangpd/test.git
$ git push -u origin master
