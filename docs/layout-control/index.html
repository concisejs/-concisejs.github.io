布局控制

ConciseJS 项目约定 src/pages/_Layout.jsx 为布局控制文件，返回一个 React 组件，并通过 props.children 渲染子组件。

布局示例​

全局布局​

src/pages/_Layout.jsx
export default function Layout(props) {
  return (
    <>
      <div>header</div>
      <div>{props.children}</div>
      <div>footer</div>
    </>
  )
}

不同的布局​

src/pages/_Layout.jsx
export default function Layout(props) {
  // 登录页不设置布局
  if (props.location.pathname === '/login') {
    return <>{props.children}</>
  } else {
    return (
      <>
        <div>header</div>
        <div>{props.children}</div>
        <div>footer</div>
      </>
    )
  }
}
