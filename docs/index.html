ConciseJS 是什么

简介#

ConciseJS 是一个关于前端业务开发的整套解决方案，为开发者提供最佳的开发体验：

  • 零配置： 自动编译并打包，仅需一个依赖即可上手开发，帮助您专注于业务（代码）开发，无需关心和配置构建工具。
  • 大量自研，简化开发： 内置路由、构建、测试、权限校验、组件库、HTTP 请求库、自定义 Hooks 库、JavaScript 状态管理库等，满足日常项目的周边需求。
  • 路由系统： 同时支持配置式路由和基于文件系统的路由，支持动态路由。
  • 开发模式： 由传统的事件驱动型开发模式转变为数据驱动型开发模式。
  • 技术栈统一： 统一团队技术栈，避免不同技术栈项目之间的切换适应，避免不同项目技术栈不兼容造成的重复开发。
  • 单元测试： 内置测试，支持单元测试及测试覆盖率分析。
  • 应用落地： 已服务了 10+ 个应用，如xx、内部组件库等。
