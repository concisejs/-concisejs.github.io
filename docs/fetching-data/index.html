获取数据

ConciseJS 规定了一种获取数据的方法，基于 axios 封装，旨在简化我们的开发工作，提升开发效率。

包含功能​

  • 默认请求配置
  • 统一传参方式
  • 响应成功及异常的统一处理
  • 简化取消请求操作
  • 支持扩展

示例​

import {ajax} from 'concisejs'

export async function login(params) {
  return ajax({
    type: 'post',
    url: '',
    data: params
  })
}

export async function getLocations(params) {
  return ajax({
    type: 'get',
    url: '',
    data: params,
    timeout: 3000 // 设置请求超时时长
  })
}

async function getList (params) {
  return ajax({
    type: 'get',
    url: '',
    cancelEnable: true // 允许取消
  })
}
const getListAjax = getList()
// 取消请求
getListAjax.__cancel__()

请求配置​

baseURL​

  • 描述：如果 ajax 的 url 是一个相对 URL ，那么 basueURL 将自动加在 url 前面
  • 类型：String
  • 默认：ConciseJS 应用的访问地址，比如 http://192.168.0.60:8080/Face/#/ ，则 baseURL 为 http://192.168.0.60:8080/Face

headers​

  • 描述：自定义请求头
  • 类型：Object
  • 默认：默认设置了两个请求头字段： Content-Type 和 Authorization ，其中， Content-Type 的值为 application/x-www-form-urlencoded;charset=UTF-8， Authorization 的值为登录时的 AUTHTOKEN 信息。

timeout​

  • 描述：请求超时的毫秒数，0 表示无超时时间
  • 类型：Number
  • 默认：0

responseType​

  • 描述：服务器响应的数据类型
  • 类型：String
  • 默认：'json'

请求参数​

type​

  • 描述：创建请求使用的方法，非必填
  • 类型：String
  • 默认：'get'

url​

  • 描述：请求服务器的 URL ，必填
  • 类型：String
  • 默认：无

data​

  • 描述：与请求一起发送的数据，非必填
  • 类型：Object
  • 默认：无

timeout​

  • 描述：请求超时时间，可覆盖默认超时时间，非必填
  • 类型：Number
  • 默认：无

cancelEnable​

  • 描述：是否允许取消请求，允许时，会暴露取消方法，非必填
  • 类型：Boolean
  • 默认：false
import {ajax} from 'concisejs'

async function getList (params) {
  return ajax({
    type: 'get',
    url: '',
    cancelEnable: true // 允许取消
  })
}
const getListAjax = getList()
// 取消请求
getListAjax.__cancel__()

响应结构​

{
  "status": 0,
  "message": "",
  "data": []
}

异常处理​

异常处理分为 http 状态异常和接口返回异常两方面。

http 状态异常​

根据 http 状态码进行判断：

  • 401 ：无权限
  • 500 ：服务器出错
  • 504 ：请求超时

接口返回异常​

根据响应结构体中的 status 字段进行判断：

  • 0 ：成功
  • 1 ：失败
