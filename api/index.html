API 概览

所属分类API类型[1]功能
基础
API		useStateHook返回一个 state，以及更新 state 的函数。
useEffectHook接收一个包含命令式、且可能有副作用代码的函数。
useRefHook返回一个可变的 ref 对象，其 .current 属性被初始化为传入的参数。
useCallbackHook返回一个记忆化的回调函数。
useMemoHook返回一个记忆化的值
useImperativeHandleHookuseImperativeHandle 可以让你在使用 ref 时自定义暴露给父组件的实例值。
useImperativeHandle 应当与 forwardRef 一起使用。
注：尽量避免使用 ref 这样的命令式代码。
forwardRefHook创建一个 React 组件，这个组件能够将其接受的 ref 属性转发到其组件树下的另一个组件中。
路由
API		LinkComponent链接组件。
NavLinkComponent<Link> 的特殊版本，当它与当前 URL 匹配时，可以为元素添加样式。
PromptComponent用于在离开页面之前提示用户。
useHistoryHook返回 history 对象。
useLocationHook返回代表当前 URL 的 location 对象。
useParamsHook获取 params 对象。
useRouteMatchHook获取当前路由的匹配信息。
获取数据
API		ajaxFunction一种获取数据的方法。
ajaxExtendFunction可扩展的获取数据的方法。
useAjaxHookajax 的 Hook 形式。
useAjaxExtendHookajaxExtend 的 Hook 形式。
状态管理
API		useDispatchHook返回 dispatch 函数的引用。
useSelectorHook允许您使用选择器功能从状态仓库中提取数据。
额外的
API		useCookieHook操作 Cookie 的 Hook 。
libsFunction常用函数库集合



名词解释：

[1] API 类型说明：Function 表示普通的 JavaScript 函数。 Hook 表示 React Hook ，它本质上也是 JavaScript 函数，与 Function 不同的是，在使用 Hook 时需要遵循两条规则，一是只能在 React 函数中调用 Hook ，不能在普通的 JavaScript 函数中调用；二是只能在最顶层使用 Hook ，不能在循环、条件或嵌套函数中调用。 Component 表示 React 组件。
