ConciseJS

基于 React 的前端应用开发框架

快速开始

专注于重要的事情

ConciseJS 帮助您专注于业务（代码）开发，无需关心和配置构建工具。
您的应用程序仅需要一个构建依赖项，以确保其所有基础部分都可以无缝协同工作，不会出现复杂的版本不匹配的情况。

大量自研，简化开发

内置路由、构建、测试、权限校验、组件库、HTTP 请求库、自定义 Hooks 库、JavaScript 状态管理库等，满足日常项目的周边需求。
自研 Webpack 插件，优化打包速度、让提示信息更加友好。

基于 React 和 Redux

以声明式编写 UI ，高效、灵活的构建用户界面，同时让您的代码更加可靠，且方便调试。
提供可预测化的状态管理，可以让您构建一致化的应用。
将应用程序的状态和逻辑集中管理，可实现强大的功能。

一分钟上手 ConciseJS

ConciseJS 让您专注于代码开发，而不是构建工具。

要创建一个名为 myApp 的项目，请运行以下命令：

npm config set registry http://192.168.0.94:7002
npx @concisejs/cli create myApp

易于维护

更新依赖通常是一项艰巨且耗时的任务。发布新版本的 ConciseJS 后，您可以使用单个命令进行升级：

npm install concisejs@latest
